# 🛡️ ReviewShield

> **Review Management System** - Automate Customer Reviews & Filter Spam

## 📌 About The Project

**ReviewShield** is a complete review management system that helps businesses:
- ✅ Automatically request reviews from customers
- 🤖 Detect and filter spam reviews 
- 📊 Track review analytics and sentiment
- 🔒 Build authentic reputation

---

## ✨ Features

### 🔐 Authentication
- User Registration & Login
- JWT Token Authentication
- Role-based Access (Admin/User)
- Secure Password Hashing

### 📝 Review Management
- Submit Customer Reviews
- Spam Detection (95%+ Accuracy)
- Sentiment Analysis (Positive/Neutral/Negative)
- Review Authenticity Scoring

### 📧 Auto Review Requests
- Generate Unique Review Links
- Automated Email Requests
- Track Request Status (Pending/Sent/Completed)
- One-click Link Copy

### 🖥️ Admin Dashboard
- Create Review Requests
- View Pending Requests
- Monitor Spam Detection Stats
- Review Analytics & Reports

### 🗄️ Database Support
- SQLite (Development)
- MySQL (Production)

---

## 🏗️ Tech Stack

### Backend
- **Python 3.10+** - Programming Language
- **FastAPI** - Web Framework
- **SQLite/MySQL** - Database
- **TextBlob** - NLP & Sentiment Analysis
- **JWT** - Authentication

### Frontend
- **HTML5** - Structure
- **CSS3** - Styling
- **JavaScript** - Interactivity
- **Font Awesome** - Icons
